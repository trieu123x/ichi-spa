# Ichi Spa - Modern Trilingual Wellness Experience

A premium, responsive landing page for **Ichi Spa**, built with a focus on tranquility, luxury, and international accessibility.

![Ichi Spa Design](https://images.unsplash.com/photo-1544161515-4af6b1d4640d?q=80&w=2070&auto=format&fit=crop)

## ✨ Highlight Features

- **🌍 Trilingual Interface**: Seamlessly switch between **Vietnamese**, **English**, and **Japanese**. All content, including the 80+ item service menu, is fully localized.
- **📱 Responsive & Modern UI**: Tailored for all devices using **Tailwind CSS v4**, featuring a softer, premium aesthetic with **Italiana** and **Montserrat** typography.
- **⚡ Interactive Service Gallery**: A dedicated Menu page with high-frequency image carousels (1.8s-2.8s) and detailed marketing copy geared towards conversion.
- **🎢 Smooth Animations**: Powered by **Framer Motion**, including scroll-triggered fade-ins, staggered grid reveals, and seamless page transitions.
- **🧭 Smart Navigation**: Sticky header with IntersectionObserver integration for dynamic active link highlighting during scroll.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 8](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Internationalization**: [i18next](https://www.i18next.com/) + `react-i18next`
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: `react-router-dom`

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/trieu123x/ichi-spa.git
   cd ichi-spa
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `/src/components`: Reusable UI components (Navbar, Hero, PriceTable, etc.)
- `/src/pages`: Main page components (Home, MenuPage)
- `/src/data`: Centralized service and price data
- `/src/locales`: i18n translation dictionaries
- `/src/i18n.js`: Internationalization configuration
